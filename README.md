# Capstone Project
Analyzed 50,000 healthcare appointment records in Power BI to identify drivers of a 50% no-show rate. Cleaned and transformed data, built an interactive dashboard, and uncovered key trends in age, lead time, and transportation to support data-driven recommendations.

# Key Insights
- Identified a 50.14% no-show ratee across the dataset \
- Found highest-risk group: ages 19-35 with 14+ day lead times \
- Discovered strong correlation between transportation barriers and missed appointments \
- Determined serveral variables (e.g., gender, medical conditions) had minimal impact \

# Tools Used
- Power BI (data cleaning, modeling, dashboard creation) \
- Data transformation and validation techniques \

- Decomposition tree for root-cause analysis \
- Trend tracking and high-risk segmentation \

# Recommendations
- Implement targeted reminder system for high-risk patients \
- Reduce long lead-time scheduling or introduce waitlist options \
- Explor transportation support program for underserved groups \
